AGES 7–10 · TINKERCAD 101

Design & Print

Learn 3D design in Tinkercad and design your own printable creation — master the basics (shapes, moving, resizing, grouping), choose a project like a keychain, game token, bag tag, or fidget spinner, then export it and watch it get 3D printed.

AGES 11–14 · ARDUINO ESSENTIALS

Build & Code

Get hands-on with real circuits and code — no experience needed. Wire your first circuit with LEDs, buttons & sensors, learn to code and upload programs to an Arduino board, then team up to build and present a mini robotics project.
